# Imgoverlay

A Vulkan and OpenGL overlay rendering images from shared memory.

Started as [MangoHud](https://github.com/flightlessmango/MangoHud) fork.

## Building

```
meson build
meson install
```

# Normal usage

```
IMGOVERLAY=1 /path/to/app
```

## Keybindings

- <kbd>Shift_R + F12</kbd>: Toggle overlay
